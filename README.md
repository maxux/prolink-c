# Pioneer DJ ProLink Protocol in C

This is an internal implementation of Pioneer DJ ProLink protocol in C,
based on personnal research and awesome work from [djl-analysis.deepsymmetry.org](https://djl-analysis.deepsymmetry.org/djl-analysis/packets.html)

Main purpose is live troubleshooting and debugging of Pioneer Gears during live events without
using graphic interface, all the work here is intended to be used on plain text console without
any external dependencies.
